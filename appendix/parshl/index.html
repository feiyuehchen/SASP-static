PARSHL is the canonical peak-picking sinusoidal analysis/synthesis program by Smith and Serra (1987). Seven-step pipeline: window → FFT → peak-detect → peak-match across frames → modify → resynthesise. The reference implementation for every SMS-style tool that came after.

PARSHL 是 Smith 與 Serra 1987 年提出的挑峰式正弦分析/合成程式。七步流水線：加窗 → FFT → 峰值偵測 → 跨 frame 匹配 → 修改 → 重合成。後續所有 SMS 風工具的範本實作。
