A stochastic process is a family of random variables indexed by time. Sections below build up the vocabulary — distribution, independence, RV, stationarity, moments — that we need to define PSDs, estimators, and their variances.

隨機過程是「以時間索引的隨機變數家族」。下面各節建立詞彙 — 機率分佈、獨立、隨機變數、平穩性、矩 — 為 PSD、估計量、估計變異數打基礎。
