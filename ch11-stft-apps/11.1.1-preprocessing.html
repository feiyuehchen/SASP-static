Stage 11-1-1 · Useful Preprocessing關卡 11-1-1 · 實用前處理SASP §11.1.1 ↗

  • High-pass filter below ~60 Hz to remove rumble that confuses harmonic matching.
  • 先用 ~60 Hz 以下高通濾掉低頻 rumble，避免擾亂諧波匹配。
  • Apply a spectral whitening / LP-inverse filter when strong formants dominate.
  • 若強烈共振峰主導頻譜，做頻譜白化 / LP 反濾波。
  • Use log-magnitude spectrum so harmonics are more uniformly weighted.
  • 用對數振幅頻譜使諧波加權更均勻。
