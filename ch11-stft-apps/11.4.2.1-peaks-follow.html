Stage 11-4-2-1 · Following Spectral Peaks關卡 11-4-2-1 · 追蹤頻譜波峰SASP §11.4.2.1 ↗

For each frame's set of peaks, match to the previous frame's tracks using nearest frequency + similar amplitude. Unmatched tracks die; unmatched peaks start new tracks.

把本 frame 的峰以最近頻率 + 相似振幅匹配到前 frame 的軌跡。未匹配軌跡收尾；未匹配峰開新軌跡。
