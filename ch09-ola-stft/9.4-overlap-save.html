Stage 9-4 · Overlap-Save Method關卡 9-4 · 重疊保留法SASP §9.4 ↗

Instead of zero-padding frames (OLA), overlap the input by $N_h-1$ between consecutive blocks and discard the first $N_h-1$ cyclic-tainted samples of each block's output. Same asymptotic cost as OLA.

OLS 不補零：輸入重疊 $N_h-1$ 樣本，每 block 輸出頭 $N_h-1$ 個循環污染樣本丟掉。漸進計算量與 OLA 相同。

OLS is simpler for fixed filters; OLA is cleaner when each frame is modified because the entire output is valid.

固定濾波器用 OLS 較簡單；逐 frame 修改用 OLA 較乾淨（整個輸出都有效）。
