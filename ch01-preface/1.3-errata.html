This 8-bit companion is an original rewrite. Its errata (typos, broken demos, incorrect explanations) are tracked at the project's GitHub issues: github.com/feiyuehchen/SASP-static/issues. If you find a bug, please open an issue — we will fix it and note it here.

Known deviations from SASP

本站與 SASP 原文的已知差異

  • Numerical demos use in-browser JavaScript FFTs (Cooley–Tukey radix-2); speed and precision differ from Matlab/Octave reference implementations.
  • 數值示範使用瀏覽器內建 JavaScript FFT（Cooley–Tukey radix-2），速度與精度與 Matlab/Octave 參考實作有別。
  • For clarity, the 8-bit recreation rounds or truncates numbers when displaying them; full-precision results are kept internally.
  • 為了易讀，顯示時會四捨五入或截斷；內部仍保留完整精度。
  • All explanatory prose is original. If any phrasing matches SASP verbatim, treat it as an accidental collision and open an issue — we will rewrite.
  • 解說文字全為原創。若發現任何段落與 SASP 原文一字不差，視為意外撞詞，請開 issue，我們會改寫。
