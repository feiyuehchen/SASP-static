Odd and even window lengths need slightly different splits. Here is a cheat sheet that always works.

奇數與偶數視窗長度的拆法略有不同。這頁是永遠正確的速查表。

Even window length $N$

偶數長度 $N$

Assume the centred window is indexed $n\in\{-N/2,\ldots,N/2-1\}$.

假設中心化視窗索引為 $n\in\{-N/2,\ldots,N/2-1\}$。

  • $w[0\ldots N/2-1]$ → buffer positions $0\ldots N/2-1$.
  • $w[0\ldots N/2-1]$ → buffer 位置 $0\ldots N/2-1$。
  • $w[-N/2\ldots-1]$ → buffer positions $M-N/2\ldots M-1$.
  • $w[-N/2\ldots-1]$ → buffer 位置 $M-N/2\ldots M-1$。
  • Zeros fill the gap $N/2\ldots M-N/2-1$.
  • 零填入 $N/2\ldots M-N/2-1$。

Odd window length $N$

奇數長度 $N$

Centred indexing $n\in\{-(N-1)/2,\ldots,(N-1)/2\}$.

中心化索引 $n\in\{-(N-1)/2,\ldots,(N-1)/2\}$。

  • $w[0\ldots (N-1)/2]$ → buffer $0\ldots(N-1)/2$.
  • $w[0\ldots(N-1)/2]$ → buffer $0\ldots(N-1)/2$。
  • $w[-(N-1)/2\ldots-1]$ → buffer $M-(N-1)/2\ldots M-1$.
  • $w[-(N-1)/2\ldots-1]$ → buffer $M-(N-1)/2\ldots M-1$。
  • Zeros fill $((N-1)/2)+1\ldots M-(N-1)/2-1$.
  • 零填入 $((N-1)/2)+1\ldots M-(N-1)/2-1$。

If you are unsure, test with a known symmetric signal (e.g. a Hann window). The FFT output should be real (imag part ≈ 0 to machine precision).

不確定時，用已知對稱訊號（例如 Hann 視窗）測試。FFT 輸出應該是實數（虛部 ≈ 機器精度）。

問：正確完成對稱實視窗的零相位補零後，$\max_k|\operatorname{Im}X[k]|$ 應該是？
