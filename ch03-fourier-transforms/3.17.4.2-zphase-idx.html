Odd and even window lengths need slightly different splits. Here is a cheat sheet that always works.

奇數與偶數視窗長度的拆法略有不同。這是永遠正確的速查表。

Even window length $N$

偶數長度 $N$

Assume the centred window is indexed $n\in\{-N/2,\ldots,N/2-1\}$. Layout:

中心化視窗索引 $n\in\{-N/2,\ldots,N/2-1\}$。排法：

  • $w[0\ldots N/2-1]$ → buffer positions $0\ldots N/2-1$.
  • $w[0\ldots N/2-1]$ → buffer 位置 $0\ldots N/2-1$。
  • $w[-N/2\ldots-1]$ → buffer positions $M-N/2\ldots M-1$.
  • $w[-N/2\ldots-1]$ → buffer 位置 $M-N/2\ldots M-1$。
  • Zeros fill the gap $N/2\ldots M-N/2-1$.
  • 零填入 $N/2\ldots M-N/2-1$。

Odd window length $N$

奇數長度 $N$

Centred indexing $n\in\{-(N-1)/2,\ldots,(N-1)/2\}$. Layout:

中心化索引 $n\in\{-(N-1)/2,\ldots,(N-1)/2\}$。排法：

  • $w[0\ldots (N-1)/2]$ → buffer $0\ldots(N-1)/2$.
  • $w[0\ldots(N-1)/2]$ → buffer $0\ldots(N-1)/2$。
  • $w[-(N-1)/2\ldots-1]$ → buffer $M-(N-1)/2\ldots M-1$.
  • $w[-(N-1)/2\ldots-1]$ → buffer $M-(N-1)/2\ldots M-1$。
  • Zeros fill $((N-1)/2)+1\ldots M-(N-1)/2-1$.
  • 零填入 $((N-1)/2)+1\ldots M-(N-1)/2-1$。

After correct zero-phase padding of a real symmetric window, the FFT imaginary part should be $\le$ machine epsilon ($\sim 10^{-15}$ in double precision). If not, you have an off-by-one in one of the index ranges.

正確完成對稱實視窗的零相位補零後，FFT 虛部應 $\le$ 機器 epsilon（雙精度約 $10^{-15}$）。不是就代表某個索引 off-by-one。

✦ 關底小魔王 ✦

問：正確完成對稱實視窗的零相位補零後，$\max_k|\operatorname{Im}X[k]|$ 應該是？

彩色方塊：$M$ 點緩衝區。黃 = 視窗右半（$w[0\ldots]$）；粉 = 左半（$w[-\cdot]$）繞到末端；灰 = 中間零段。對稱輸入 → 緩衝區對 $n=0$（mod $M$）對稱 → FFT 輸出為實數。
