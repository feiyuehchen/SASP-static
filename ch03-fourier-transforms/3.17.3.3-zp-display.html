Stage 3-17-3-3 · ZP for Spectral Displays 關卡 3-17-3-3 · 以補零改善頻譜顯示 SASP §3.17.3.3 ↗

For a plot you can read with the eye, zero-pad so the FFT is 4× to 8× the window length.

要畫出眼睛看得清楚的頻譜圖，補零讓 FFT 長度是視窗的 4× 到 8×。

Without padding, the FFT samples are spaced at $2\pi/N$ — roughly the Hann window's main-lobe half-width. Only two samples across a lobe, so every peak looks like a jagged zigzag. Padding to $M=4N$ or $8N$ lays down 4 or 8 samples inside the same lobe and the curve becomes smooth enough to read.

不補零時，FFT 樣本間距 $2\pi/N$ — 大約是 Hann 視窗主瓣半寬。 一個主瓣只有兩個樣本，每個峰像鋸齒。補零到 $M=4N$ 或 $8N$，同一主瓣內有 4 或 8 個樣本， 曲線就能一眼讀懂。

FFT cost scales as $M\log M$. Going from $N$ to $8N$ multiplies the work by about $11\times$, which is usually fine for plots but might hurt real-time applications. Use the minimum $M$ you need for the decision you are making (display, peak picking, etc.).

FFT 計算量 $M\log M$。從 $N$ 變 $8N$ 約 $11\times$，畫圖沒差、即時應用要注意。依使用目的挑剛好夠用的 $M$。

PADDING CHEAT SHEET
Display: $M = 4N\ldots 8N$ (smooth curves).
Peak picking: $M = 2N$ + quadratic interpolation (see §3.17.3.4).
Convolution: $M \ge N_x + N_h - 1$ (avoid circular wrap).
No change of behaviour: $M = N$ (nothing happens). 顯示：$M = 4N\ldots 8N$（曲線平滑）。
找波峰：$M = 2N$ + 拋物線內插（見 §3.17.3.4）。
卷積：$M \ge N_x + N_h - 1$（避免循環 wrap）。
不改變行為：$M = N$（沒發生任何事）。
