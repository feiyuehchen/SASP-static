Stage 12-7-2 · PR Cosine-Modulated FBs關卡 12-7-2 · 完美重建餘弦調變組SASP §12.7.2 ↗

PR cosine-modulated banks (MLT / MDCT) get exact PR by using two overlapping modulated cosines per band: an odd-DCT analysis and a shifted odd-DCT synthesis. Foundation of mp3, AAC, Vorbis, Opus.

完美重建餘弦調變組（MLT / MDCT）每帶用兩個重疊的調變餘弦：奇偶 DCT 分析 + 位移的奇偶 DCT 合成，獲得精確 PR。mp3、AAC、Vorbis、Opus 都以它為基礎。
