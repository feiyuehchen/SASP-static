Stage 5-10-4 · Chebyshev FIR via Linear Programming 關卡 5-10-4 · 用 LP 設計 Chebyshev FIR SASP §5.10.4 ↗

Discretise the minimax problem on a dense frequency grid and solve an LP. Exactly the trick we used for window design — same code works, just change the target $H_d(\omega)$.

把 minimax 問題在密格點上離散化、解 LP。與 §4.13 視窗設計是同一招 — 換個目標 $H_d(\omega)$ 就能用。

Slightly slower than Remez on the same problem, but handles non-standard constraints (non-negativity of $|H|$, combined bands, custom weights) that Parks-McClellan can't.

比 Remez 慢一點，但能處理非標準約束（$|H|$ 非負、組合頻帶、自訂權重），Parks-McClellan 做不到。
