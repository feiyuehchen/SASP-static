% Matlab: ideal Hilbert impulse response
N = 64;                       % even Type IV
n = -(N/2-1):(N/2);
h = (2/pi) * (mod(n,2) ./ n);  % zero for even n; 2/(pi n) for odd
h(n==0) = 0;                  % handle DC manually

% Window it
w = kaiser(N, 6);
h_fir = h(:) .* w;

Matlab's firpm with 'Hilbert' type does this too, using Parks-McClellan for optimal minimax error.
