Stage 5-5-1 · Matlab Support for the Window Method 關卡 5-5-1 · 視窗法的 Matlab SASP §5.5.1 ↗ 

% Matlab: fir1 does the whole job
h = fir1(N-1, wc/pi, hann(N));         % lowpass
h = fir1(N-1, [wp1 wp2]/pi*2, 'bandpass', blackman(N));

% Python / scipy
from scipy.signal import firwin
h = firwin(N, wc/pi, window='hann')

fir1 picks the window for you if you don't specify one. You can also pass your own custom window from World 4.

fir1 若不指定視窗會自己選一個。也可以把第 4 世界的自訂視窗丟給它。
