Stage 4-13-11 · L1 Derivative Objective 關卡 4-13-11 · L1 導數目標 SASP §4.13.11 ↗

$\min\sum_n|w[n+1]-w[n]|$ — the total variation of $w$. Unlike L∞ which penalises the worst slope, L1 penalises the sum — so it prefers piecewise-constant shapes where most differences are zero and the variation is concentrated in a few large jumps.

$\min\sum_n|w[n+1]-w[n]|$ — $w$ 的總變差。L∞ 懲罰最差斜率，L1 懲罰總和 — 因此偏好分段常數：多數差為零、變化集中在少數大跳變。

L1 DERIVATIVE LP
$$\begin{aligned} &\min_{w,t}\;\sum_n t[n]\\ &\text{s.t.}\;-t[n]\le w[n+1]-w[n]\le t[n]\\ &\phantom{\text{s.t.}\;}|W(\omega_i)|\le\delta\;\text{(side-lobe ceiling)}\\ &\phantom{\text{s.t.}\;}\sum_n w[n]=1 \end{aligned}$$

Auxiliary variables $t[n] \ge |w[n+1]-w[n]|$ linearise the absolute value. Sum of $t$ becomes the linear cost. At optimum, $t[n] = |w[n+1]-w[n]|$ exactly.

輔助變數 $t[n] \ge |w[n+1]-w[n]|$ 線性化絕對值。$t$ 之和成線性成本。最優處 $t[n] = |w[n+1]-w[n]|$ 嚴格成立。

When L1 wins

L1 何時較佳

  • Sparse-approximation pedagogy: shows the L1-promotes-sparsity pattern that recurs in compressive sensing, LASSO, basis pursuit.
  • 稀疏逼近教學：呈現 L1 推稀疏的模式，在壓縮感測、LASSO、基追蹤中反覆出現。
  • Block-edge detection: a piecewise-constant window overlap-adds into discrete blocks naturally.
  • 區塊邊緣偵測：分段常數視窗自然地以離散區塊重疊相加。
  • Quantised-window memory savings: piecewise-constant ↔ run-length encoded → smaller storage.
  • 量化視窗節省記憶：分段常數 ↔ 連長編碼 → 更小儲存。

L1 windows have step-discontinuities — looking at the time-domain you'll see flat plateaus connected by sudden jumps. Bad for audio: those jumps generate slow $1/\omega$ spectral decay, exactly opposite of what we usually want. Stick with L∞ or sidelobe-depth objectives for spectrum analysis.

L1 視窗有階梯不連續 — 時域為平台與突變連接。音訊不利：突變產生 $1/\omega$ 慢衰減，與一般需求相反。頻譜分析仍用 L∞ 或旁瓣深度目標。

L1 ⇒ sparsity, L∞ ⇒ uniformity. The same principle that makes LASSO favour few non-zero coefficients makes L1-derivative windows piecewise-constant. Different norm, different solution geometry.

L1 ⇒ 稀疏、L∞ ⇒ 均勻。使 LASSO 偏好少量非零係數的同一原則，使 L1 導數視窗成分段常數。不同範數、不同解幾何。
