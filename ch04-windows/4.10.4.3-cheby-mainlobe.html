Stage 4-10-4-3 · Main-Lobe Width 關卡 4-10-4-3 · 主瓣寬度 SASP §4.10.4.3 ↗

HALF MAIN-LOBE WIDTH (ω)
$$\omega_0\;=\;2\cos^{-1}\!\left(\frac{1}{\beta}\right)$$

For large attenuation $A$, $\beta\approx\tfrac{1}{N-1}\ln(2\cdot 10^{A/20})$, and the main-lobe width is roughly proportional to $A/N$. So doubling $N$ halves the main lobe for the same side-lobe depth.

大 $A$ 時，$\beta\approx\tfrac{1}{N-1}\ln(2\cdot 10^{A/20})$，主瓣寬大致正比於 $A/N$。 即：固定旁瓣深度下，$N$ 變 2 倍，主瓣寬減半。
