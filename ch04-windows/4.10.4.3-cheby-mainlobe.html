Stage 4-10-4-3 · Main-Lobe Width 關卡 4-10-4-3 · 主瓣寬度 SASP §4.10.4.3 ↗

The main lobe is the region where $|\beta\cos(\omega/2)| > 1$. Solving the boundary gives the half-width $\omega_0$ — from the centre $\omega=0$ to the first null.

主瓣即 $|\beta\cos(\omega/2)| > 1$ 的區域。邊界方程給出半寬 $\omega_0$ — 由中央 $\omega=0$ 到首零。

HALF MAIN-LOBE WIDTH (RAD)
$$\omega_0\;=\;2\cos^{-1}\!\left(\frac{1}{\beta}\right)$$

Large-$A$ approximation

大 $A$ 近似

For large $A$, $\beta\approx\tfrac{1}{N-1}\ln\!\bigl(2\cdot 10^{A/20}\bigr)$, so the main-lobe width is roughly proportional to $A/N$. Doubling $N$ at fixed $A$ halves the main lobe; doubling $A$ at fixed $N$ roughly doubles the main lobe.

大 $A$ 時 $\beta\approx\tfrac{1}{N-1}\ln\!\bigl(2\cdot 10^{A/20}\bigr)$，主瓣寬大致正比於 $A/N$。$A$ 固定，$N$ 加倍 → 主瓣減半；$N$ 固定，$A$ 加倍 → 主瓣加倍。

LARGE-A APPROXIMATION
$$\omega_0\;\approx\;\frac{A\ln 10\;+\;\ln 2}{10\,(N-1)}\quad\text{(in radians)}$$

$N=64$ 時比較表

Minimum main lobe per dB of attenuation. No other window achieves smaller main lobe at the same $A$ — that's the minimax theorem paying off in design units.

每 dB 衰減的最小主瓣。同 $A$ 下無其他視窗有更窄主瓣 — 這就是 minimax 定理轉化成設計值。
