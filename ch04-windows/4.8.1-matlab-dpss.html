Stage 4-8-1 · Matlab for the DPSS Window 關卡 4-8-1 · DPSS 的 Matlab SASP §4.8.1 ↗

Matlab's dpss(N, NW) returns the matrix of all $N$ prolate sequences sorted by eigenvalue (concentration). Usually you only need the first one or two; the rest are for multi-taper PSD (§7 / Thomson's method).

Matlab dpss(N, NW) 回傳所有 $N$ 個扁球體序列（依特徵值/集中度排序）。通常只用前 1 或 2 個；其餘給多 taper PSD 用（§7 Thomson 法）。

% Matlab / Octave (Signal Processing Toolbox)
N  = 512;
NW = 4;                         % time-bandwidth product
[E, V] = dpss(N, NW);           % E: N×K tapers, V: concentration eigenvalues

w = E(:, 1);                    % primary taper (V(1) closest to 1)

% First few tapers — how multi-taper uses them
K = 2 * NW - 1;                 % # of usable tapers
tapers = E(:, 1:K);

% Python / scipy
from scipy.signal.windows import dpss
w = dpss(N, NW)                 % first taper only
tapers = dpss(N, NW, Kmax=K)    % first K tapers

Typical $NW$ values

常見 $NW$ 值

$NW = 2.5$: 4 usable tapers, ~$-55$ dB side-lobes, moderate main-lobe.
$NW = 4$: 7 tapers, $-80$ dB side-lobes, wider main lobe.
$NW = 6$: 11 tapers, $-100$ dB, quite wide main lobe.
Pick $NW$ according to the number of tapers you need vs. frequency resolution you can afford. $NW = 2.5$：4 個可用 taper，旁瓣約 $-55$ dB，主瓣中等。
$NW = 4$：7 tapers，旁瓣 $-80$ dB，主瓣較寬。
$NW = 6$：11 tapers，$-100$ dB，主瓣很寬。
依「所需 taper 數 vs. 可接受解析度」選 $NW$。

Requires the Signal Processing Toolbox in Matlab (or scipy in Python). For a closed-form substitute see Kaiser (§4.9), which approximates DPSS with $\beta \approx \pi NW$.

Matlab 需 Signal Processing Toolbox（Python 需 scipy）。想要閉式替代見 Kaiser（§4.9），$\beta \approx \pi NW$ 近似 DPSS。

dpss returns a matrix, not a vector. The columns are the orthogonal tapers sorted by concentration. Use column 1 for classic windowing; use all $2NW-1$ for multi-taper PSD.

dpss 回傳矩陣、非向量。各行為依集中度排序的正交 tapers。單視窗取第 1 行；多 taper PSD 用前 $2NW-1$ 行。

問：$N=512, NW=4$，dpss 回傳多少個特徵值接近 1 的可用 tapers？

Lab · Multiple Kaiser "tapers" at the same β

實驗 · 同一 β 的多個 Kaiser「tapers」

Simulated multi-taper view — shown with Kaiser windows shifted in the time axis. Real DPSS tapers have orthogonal shapes (each has a different number of zero-crossings), which is the key property that makes averaged periodograms independent. Kaiser alone doesn't give this orthogonality, but the lab here gives a feel for "many overlapping similar windows".

多 taper 的模擬視覺 — 用時間軸上位移的 Kaiser 視窗表現。真正的 DPSS tapers 形狀互相正交（每個零交越次數不同），這是平均 periodogram 獨立性的關鍵。Kaiser 不具備此正交性，但 lab 給你「許多相似重疊視窗」的感覺。
