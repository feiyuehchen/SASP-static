% Matlab / Octave (signal processing toolbox)
NW = 4;                   % time-bandwidth product
[E, V] = dpss(N, NW);     % E columns = orthogonal tapers
w = E(:,1);               % usually use the first one

% Python / scipy
from scipy.signal.windows import dpss
w = dpss(N, NW)           % returns the first taper

NW is the time-bandwidth product (half-width of the main lobe × $N$). Typical values 2.5–4. Larger NW pushes sidelobes down further.

NW 是時間頻寬乘積（主瓣半寬 × $N$）。常用 2.5–4。NW 越大，旁瓣壓越深。
