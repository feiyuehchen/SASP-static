Stage 4-12-2 · General Rule 關卡 4-12-2 · 通則 SASP §4.12.2 ↗

Pick the window that makes your worst-case interferer smallest in the band you care about, subject to the main-lobe and reconstruction constraints you can afford.

在可接受的主瓣寬與重建約束下，選能讓你最關心的那一帶最糟干擾變最小的視窗。

Each named window = one Pareto point

每個具名視窗 ＝ 柏拉圖前緣上一點

Every window studied in this world represents a specific trade on the two main axes — main-lobe width and side-lobe depth — plus auxiliary constraints like OLA, smoothness, endpoint value. Moving one axis always costs you the other.

本世界研讀的每個視窗，都代表主瓣寬與旁瓣深這兩軸上某個取捨點，加上 OLA、平滑、端點值這類附加約束。移動一軸必定在另一軸付出代價。

PARETO CHEAT SHEET
Rectangular: narrowest main lobe, worst side-lobe (−13 dB).
Hann / Hamming: standard audio-grade, moderate both.
Blackman / B-Harris / Nuttall: deeper sides, wider main.
Kaiser / DPSS / Dolph-Cheby: parametric — pick your axis.
MLT sine / Vorbis: optimised for squared-OLA codecs.
LP-designed (§4.13): any Pareto point, pay with compute. 矩形：主瓣最窄、旁瓣最差（-13 dB）。
Hann / Hamming：音訊標準，兩軸中等。
Blackman / B-Harris / Nuttall：旁瓣深、主瓣寬。
Kaiser / DPSS / Dolph-Cheby：參數化 — 選軸。
MLT 正弦 / Vorbis：為平方 OLA 編碼器最佳化。
LP 設計（§4.13）：任何柏拉圖點，代價是運算。

The decision checklist

決策清單

  1. What's the interferer? Strong neighbour tone → narrow main lobe wins. Broadband noise → low side-lobes win.
  2. 干擾是什麼？強鄰音 → 窄主瓣勝；寬頻雜訊 → 低旁瓣勝。
  3. Where is the interferer? Near the target → Dolph-Chebyshev plateau. Far away → Hamming-like roll-off.
  4. 干擾在哪？靠近目標 → Dolph-Chebyshev 平台；離目標遠 → Hamming 類滾降。
  5. Reconstruction needed? Yes → Hann, MLT sine, Vorbis. No → any analysis-side window.
  6. 需要重建嗎？是 → Hann、MLT 正弦、Vorbis。否 → 任何解析端視窗。
  7. Latency budget? Low → short $N$. Long OK → everything opens up.
  8. 延遲預算？低 → 短 $N$。可接受長 → 任何視窗皆可。

Window design is Pareto frontier selection. There is no universal "best" window — only "best given these application constraints." The remaining §4.13 sub-sections show how to compute an arbitrary Pareto point by LP, when catalog windows aren't quite right.

視窗設計是柏拉圖前緣選點。沒有通用「最佳」視窗 — 只有「這組應用約束下的最佳」。§4.13 教你在目錄視窗不夠合身時，如何用 LP 計算任意柏拉圖點。
