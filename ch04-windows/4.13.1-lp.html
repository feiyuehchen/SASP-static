Stage 4-13-1 · Linear Programming 關卡 4-13-1 · 線性規劃 SASP §4.13.1 ↗

Minimise a linear cost subject to linear inequalities over a real vector. Geometry: the feasible region is a polytope (intersection of half-spaces), and the optimum sits at a vertex. Numerical tooling: Matlab linprog, scipy linprog, GLPK, CVXPY, CPLEX — every numerical environment has a polynomial-time solver.

在線性不等式約束下最小化一個線性成本。幾何：可行區為多胞形（半空間交集），最優落在頂點。數值工具：Matlab linprog、scipy linprog、GLPK、CVXPY、CPLEX — 每個數值環境都有多項式時間求解器。

LP STANDARD FORM
$$\min_{x}\ c^{\top}x \quad\text{s.t.}\quad Ax\le b,\;\; x\ge 0$$

The window-design trick

視窗設計的巧思

Encode "main-lobe peak $=1$, side-lobes $\le \delta$, $\delta$ minimal" as linear statements. Treat $\delta$ as an extra variable alongside $w$, with cost vector $c=(0,0,\dots,0,1)$ picking out $\delta$. Minimising $\delta$ is the LP — and the optimal $w$ falls out as part of the solution.

把「主瓣峰 $=1$、旁瓣 $\le \delta$、$\delta$ 最小」寫成線性式。把 $\delta$ 當 $w$ 旁的額外變數，成本向量 $c=(0,0,\dots,0,1)$ 只挑出 $\delta$。極小化 $\delta$ 就是這個 LP，最優 $w$ 會附帶解出。

Solver quick reference

求解器速查

% Matlab / Octave
[x, fval] = linprog(c, A, b, Aeq, beq, lb, ub);

# Python (scipy)
from scipy.optimize import linprog
result = linprog(c, A_ub=A, b_ub=b, A_eq=Aeq, b_eq=beq,
                 bounds=list(zip(lb, ub)), method='highs')
x = result.x

# Python (CVXPY, more readable)
import cvxpy as cp
w = cp.Variable(N)
delta = cp.Variable()
prob = cp.Problem(cp.Minimize(delta),
    [A @ w - b <= delta, -(A @ w - b) <= delta,  # |Aw - b| ≤ δ
     cp.sum(w) == 1])                             # DC constraint
prob.solve()

LP is the universal glue for window design. Any constraint you can express as "linear-in-$w$ must be $\le$ something" (side-lobes, main-lobe shape, symmetry, OLA) drops straight into the formulation. The solver finds the globally optimal combination.

LP 是視窗設計的通用膠水。任何能寫成「$w$ 線性函數 $\le$ 某值」的約束（旁瓣、主瓣形狀、對稱、OLA）都能直接放入。求解器給出全域最優組合。
