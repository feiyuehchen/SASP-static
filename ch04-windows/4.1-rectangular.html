Stage 4-1 · Rectangular Window 關卡 4-1 · 矩形視窗 SASP §4.1 ↗

Slice out a run of $N$ samples; set everything else to zero. That is the rectangular window — the one you implicitly use whenever you "take an FFT of the first 1024 samples."

取出連續 $N$ 個樣本、其餘歸零。這就是矩形視窗 — 當你「對前 1024 個樣本做 FFT」時，其實就是它。

DEFINITION
$$w_R[n]=\begin{cases}1,& 0\le n DTFT (ASYMPTOTIC DIRICHLET)
$$W_R(\omega)=e^{-j\omega(N-1)/2}\,\frac{\sin(N\omega/2)}{\sin(\omega/2)}$$

Key numbers

關鍵數字

  • Main-lobe width: $4\pi/N$ (zero-to-zero).
  • 主瓣寬度：$4\pi/N$（零點到零點）。
  • First side-lobe level: about $-13$ dB below the main lobe — quite loud.
  • 第一旁瓣高度：約比主瓣低 $13$ dB — 相當高。
  • Side-lobe roll-off: $6$ dB/octave (slow). That is the price for having the narrowest possible main lobe.
  • 旁瓣滾降：$6$ dB/八度（很慢）。這是擁有最窄主瓣的代價。

The rectangular window is the best choice only when the signal is exactly $N$ samples of a periodic signal whose period divides $N$. Otherwise the huge side lobes will smear other frequencies. Almost always, you should reach for Hann or Hamming instead.

矩形視窗只在「訊號剛好是 $N$ 個樣本、週期整除 $N$」時是最佳選擇。 否則巨大的旁瓣會把別的頻率糊掉。實務上通常改用 Hann 或 Hamming。
