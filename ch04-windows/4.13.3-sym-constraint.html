Stage 4-13-3 · Symmetric Constraint 關卡 4-13-3 · 對稱約束 SASP §4.13.3 ↗

A real symmetric window of length $N$ has only $\lceil N/2\rceil$ distinct values — $w[n]=w[N-1-n]$ ties the pairs. Adding this as an equality constraint halves the variable count and gives you a zero-phase spectrum for free.

長度 $N$ 的實對稱視窗只有 $\lceil N/2\rceil$ 個獨立值 — $w[n]=w[N-1-n]$ 把兩兩綁定。加等式約束使變數減半，並附送零相位頻譜。

SYMMETRY EQUALITIES
$$w[n]\;=\;w[N-1-n],\qquad n=0,\dots,\lfloor (N-1)/2\rfloor$$

Two ways to enforce

兩種實現方式

Option A — equality constraints: Add $\lfloor N/2\rfloor$ linear equalities $w[n]-w[N-1-n]=0$ to the LP. Full $w$ variable kept; solver figures out symmetry by itself.

選項 A — 等式約束：LP 加入 $\lfloor N/2\rfloor$ 條線性等式 $w[n]-w[N-1-n]=0$。保留完整 $w$ 變數，由求解器自行滿足對稱。

Option B — change of variables: Use only $h=w[0..\lceil N/2\rceil-1]$ and build the symmetric $w$ post-solve. Spectrum simplifies to $W(\omega)=\sum_{m=0}^{M} h[m]\cos(\omega\cdot(m-(N-1)/2))$ — half the cosines, half the LP matrix.

選項 B — 變數代換：只用 $h=w[0..\lceil N/2\rceil-1]$，求解後再鏡像建 $w$。頻譜簡化為 $W(\omega)=\sum_{m=0}^{M} h[m]\cos(\omega\cdot(m-(N-1)/2))$ — cos 項減半、LP 矩陣減半。

Symmetric real → spectrum is real and even → phase is identically zero (or ±π where magnitude crosses zero). No phase distortion when applied to an FFT analysis (§3.5.2).

實對稱 → 頻譜為實且偶 → 相位恆為 0（或振幅過零時 ±π）。用於 FFT 解析時沒有相位失真（§3.5.2）。

Symmetry is free speed. Halves the LP in both variable count and row count, and zero-phase spectrum comes along as a bonus. Virtually every audio window should be symmetric unless you have a specific reason not to.

對稱＝免費加速。LP 變數與列數雙雙減半，零相位頻譜附贈。除非有特殊理由，所有音訊視窗都應對稱。
