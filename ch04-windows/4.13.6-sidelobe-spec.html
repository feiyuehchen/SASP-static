Stage 4-13-6 · Sidelobe Specification 關卡 4-13-6 · 旁瓣規格 SASP §4.13.6 ↗

Instead of one flat floor, allow the side-lobe envelope to depend on $\omega$: deeper where it matters, relaxed where it does not.

不必所有旁瓣都同一高度 — 可以讓旁瓣包絡隨 $\omega$ 變化：在重要頻段更深，其他地方可放寬。

$$|W(\omega_i)|\le\delta\cdot g(\omega_i)\quad \forall\omega_i\in\text{stopband}$$

Weight function $g(\omega)$: set small where you must suppress interference, large where you don't care. Same LP, different right-hand side.

權重函數 $g(\omega)$：在必須壓干擾的頻段取小值、其他取大值。同一個 LP，只是右端不同。
