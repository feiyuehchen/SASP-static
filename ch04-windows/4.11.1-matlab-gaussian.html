% Matlab — uses alpha = 1/(2*sigma)
w = gausswin(N, alpha);

% Manual
n = (0:N-1)';
M = (N-1)/2;
sigma = 0.4;
w = exp(-0.5 * ((n - M) / (sigma * M)).^2);

Matlab's parameter is the reciprocal: $\alpha=\tfrac{1}{2\sigma}$. A larger alpha → tighter taper → wider spectrum.

Matlab 的參數是倒數關係 $\alpha=\tfrac{1}{2\sigma}$。alpha 大 → 收尾緊 → 頻譜寬。
