Stage 4-11-1 · Matlab for Gaussian 關卡 4-11-1 · Gaussian 的 Matlab SASP §4.11.1 ↗

Matlab's built-in gausswin uses $\alpha = 1/(2\sigma)$ — the reciprocal of the $\sigma$ parameter SASP prefers. Larger $\alpha$ → tighter taper → wider spectrum. Easy to confuse, so either spell it out or build manually.

Matlab 內建 gausswin 用的是 $\alpha = 1/(2\sigma)$ — 與 SASP 偏好的 $\sigma$ 互為倒數。$\alpha$ 大 → 收尾緊 → 頻譜寬。容易混淆，不是寫清楚就是自己手寫。

% Built-in: alpha = 1/(2*sigma)
N = 256;  alpha = 2.5;           % alpha=2.5 → sigma=0.2
w1 = gausswin(N, alpha);

% Manual, parametrised by sigma
function w = gauss_window(N, sigma)
  n = (0:N-1)';
  M = (N-1)/2;
  w = exp(-0.5 * ((n - M) ./ (sigma * M)).^2);
end

sigma = 0.4;
w2 = gauss_window(N, sigma);

% Sanity: they should agree with alpha = 1/(2*sigma)
max(abs(w1 - gauss_window(N, 0.2)))  % ≈ 1e-16

Typical σ values

常見 σ 值

σ = 0.5: very loose taper, wide time bump → narrow spectrum. Good for fine frequency resolution.
σ = 0.4: the SASP default. Balanced — good main-lobe/side-lobe trade.
σ = 0.3: tighter taper, main-lobe $\approx 6\pi/N$, side-lobes $\le -65$ dB.
σ = 0.2: very tight, side-lobes below $-95$ dB but main-lobe becomes $> 10\pi/N$. σ = 0.5：收尾極鬆、時域寬 → 頻域窄。適合細緻頻率解析度。
σ = 0.4：SASP 預設。主瓣/旁瓣取捨均衡。
σ = 0.3：稍緊，主瓣約 $6\pi/N$、旁瓣 $\le -65$ dB。
σ = 0.2：極緊，旁瓣低於 $-95$ dB，但主瓣變成 $> 10\pi/N$。

σ parameterises the uncertainty trade directly. Think "time support in units of half-window" — $\sigma=0.4$ means the Gaussian's $\pm 1\sigma$ extends to $\pm 40\%$ of the half-window.

σ 直接量化測不準取捨。想成「以半視窗為單位的時間支撐」— $\sigma=0.4$ 代表 $\pm 1\sigma$ 範圍延伸到半視窗的 $\pm 40\%$。
