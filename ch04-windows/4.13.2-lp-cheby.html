Stage 4-13-2 · LP Formulation of Chebyshev Design 關卡 4-13-2 · LP 描述 Chebyshev 設計 SASP §4.13.2 ↗

Discretise $\omega$ on a dense grid in the stopband. For symmetric real $w$, $W(\omega_i)=\sum_n w[n]\cos(\omega_i n)$ is linear in $w$. Require $|W(\omega_i)|\le\delta$ for all stopband $\omega_i$, $W(0)=1$, and minimise $\delta$. The solver returns Dolph-Chebyshev for free — with any extra constraint of your choice.

在阻帶以密格點取 $\omega$。對稱實窗 $w$ 下 $W(\omega_i)=\sum_n w[n]\cos(\omega_i n)$ 對 $w$ 線性。要求所有阻帶 $\omega_i$ 的 $|W(\omega_i)|\le\delta$、$W(0)=1$，並最小化 $\delta$。求解器會自動吐出 Dolph-Chebyshev — 還能隨時加入任何額外約束。

THE PROBLEM
$$\begin{aligned}&\min_{w,\delta}\;\delta\\ &\text{s.t.}\; W(\omega_i)\le\delta, \; \omega_i\in\text{stopband}\\ &\phantom{\text{s.t.}\;} -W(\omega_i)\le\delta\\ &\phantom{\text{s.t.}\;} W(0)=\sum_n w[n]=1\\ &\phantom{\text{s.t.}\;} w[n]=w[N-1-n] \text{ (symmetry)}\end{aligned}$$

The first two inequality sets encode $|W(\omega_i)|\le\delta$ (absolute value becomes two one-sided bounds). The equality $W(0)=1$ prevents the trivial $w=0$ solution. Symmetry halves the variable count.

前兩組不等式編碼 $|W(\omega_i)|\le\delta$（絕對值化為兩條單邊界）。等式 $W(0)=1$ 防止 $w=0$ 平凡解。對稱使變數減半。

Reference scipy implementation

scipy 參考實作

import numpy as np
from scipy.optimize import linprog

N, A_dB = 64, 60
omega_p = 5*np.pi/N                       # passband half-width (main-lobe edge)
M_grid = 400                              # stopband grid points
omega_s = np.linspace(omega_p, np.pi, M_grid)

# variables: w[0..N-1] + delta → x ∈ R^(N+1)
# symmetric real spectrum: W(ω) = Σ w[n] cos(ω(n - (N-1)/2))
n_sh = np.arange(N) - (N-1)/2
C = np.cos(np.outer(omega_s, n_sh))       # shape (M, N)
# |C·w| ≤ δ  →  C·w - δ·1 ≤ 0  and  -C·w - δ·1 ≤ 0
A_ub = np.vstack([
    np.hstack([C,  -np.ones((M_grid, 1))]),
    np.hstack([-C, -np.ones((M_grid, 1))]),
])
b_ub = np.zeros(2*M_grid)
# W(0) = Σ w[n] = 1
A_eq = np.hstack([np.ones((1, N)), [[0]]])
b_eq = np.array([1.0])
# objective: minimise δ → c = (0,0,...,0,1)
c = np.zeros(N+1); c[-1] = 1

res = linprog(c, A_ub=A_ub, b_ub=b_ub, A_eq=A_eq, b_eq=b_eq,
              bounds=[(None,None)]*N + [(0,None)], method='highs')
w, delta = res.x[:N], res.x[-1]
print(f"side-lobe level = {20*np.log10(delta):.2f} dB")

Replace the equality $W(0)=1$ with $W(\omega_i)\ge 1-\delta_p$ for $\omega_i$ in the passband, and add both $\delta_s$ (stopband) and $\delta_p$ (passband) to the cost → you get general Parks-McClellan-style minimax FIR design. Chebyshev window is just the special case $\delta_p = 0$.

把等式 $W(0)=1$ 換成「通帶 $\omega_i$ 上 $W(\omega_i)\ge 1-\delta_p$」，成本加入通帶 $\delta_p$ 與阻帶 $\delta_s$ → 這就是一般的 Parks-McClellan minimax FIR 設計。Chebyshev 視窗是 $\delta_p=0$ 特例。

LP reproduces Dolph-Chebyshev and generalises it. Same formulation → any constraint you add yields a new optimal window. The closed-form Chebyshev formula is an LP solution with the most minimal constraint set.

LP 重現且廣義化 Dolph-Chebyshev。同一框架下加任何約束都會產生新最優視窗。閉式 Chebyshev 只是約束最少時的 LP 解。
