Stage 4-13-8-1 · Remez Convergence 關卡 4-13-8-1 · Remez 收斂 SASP §4.13.8.1 ↗

Remez exchange terminates in finitely many iterations and always reaches the global optimum of the minimax sidelobe problem. Both facts follow from the Chebyshev alternation theorem.

Remez 交換有限步必停止，並必達 minimax 旁瓣問題的全域最優。兩點皆由 Chebyshev 交替定理得證。

Why convergence is finite

為何有限步必收斂

Each iteration's $\delta_k$ is non-decreasing — the new extrema by construction yield a $\delta_{k+1}\ge\delta_k$ — and bounded above by the true optimum. Bounded monotone sequence → it converges. Because we're updating an integer-indexed extremal set drawn from a finite grid, exact convergence is reached in finitely many swaps.

每次迭代 $\delta_k$ 非遞減 — 新極值依結構必滿足 $\delta_{k+1}\ge\delta_k$ — 且有真最優上界。有界單調 → 收斂。因為更新的是有限格上的整數索引極值集合，有限次交換即達精確收斂。

CONVERGENCE BOUND
$$\delta_k\;\nearrow\;\delta^*\quad\text{with}\quad\bigl|\delta^* - \delta_k\bigr|\;\le\; C\rho^k,\quad\rho<1$$

Geometric (linear) convergence: error halves every fixed number of steps. Typical $\rho \approx 0.3$ - $0.5$, so 5-15 iterations suffice for double-precision accuracy.

幾何（線性）收斂：誤差每固定步數減半。典型 $\rho \approx 0.3$ - $0.5$，5-15 次迭代達雙精度。

Per-iteration cost

每次迭代成本

  • One linear solve on $\lceil N/2\rceil+1$ extremal points → $O(N^3)$ if dense, faster if structured.
  • 一次線性求解於 $\lceil N/2\rceil+1$ 個極值點 → 一般 $O(N^3)$，有結構時更快。
  • One grid sweep for new extrema → $O(N\cdot M)$ where $M$ = grid size.
  • 一次格點掃描找新極值 → $O(N\cdot M)$，$M$ 為格點數。
  • No factorisations carry over; each iteration is independent.
  • 不繼承前一步分解；每次迭代獨立。

Numerical issues at very high $A$ ($\gg 120$ dB): coefficient matrix $\cosh(N\cosh^{-1}\beta)$ overflows. Use double-precision long-double or specialised stable forms (cf. PMS or McClellan-Parks 1972). For audio $A\le 120$ dB this never bites.

極高 $A$（$\gg 120$ dB）時數值問題：係數矩陣 $\cosh(N\cosh^{-1}\beta)$ 溢位。需 long-double 或穩定變形（見 PMS 或 McClellan-Parks 1972）。音訊 $A\le 120$ dB 不會遇到。

Geometric convergence + alternation theorem. Together they make Remez the production-grade algorithm — provably optimal, polynomially fast, and trivial to detect convergence (just watch $\delta_k$ stop changing).

幾何收斂 + 交替定理。兩者使 Remez 成為實用級演算法 — 可證最優、多項式時間、收斂偵測簡單（看 $\delta_k$ 停止變化即可）。
