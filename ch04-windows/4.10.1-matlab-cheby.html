% Matlab
A_dB = 60;                     % desired side-lobe attenuation
w = chebwin(N, A_dB);

% Python (scipy)
from scipy.signal.windows import chebwin
w = chebwin(N, at=60)

Matlab's chebwin takes the side-lobe attenuation in dB directly, so no need to invert the $\beta$ formula by hand.

Matlab 的 chebwin 直接吃 dB 衰減，不必手算 $\beta$。
