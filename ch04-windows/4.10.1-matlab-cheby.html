Stage 4-10-1 · Matlab for Dolph-Chebyshev 關卡 4-10-1 · Dolph-Chebyshev 的 Matlab SASP §4.10.1 ↗

Matlab's chebwin takes the side-lobe attenuation in dB directly — no need to invert the $\beta$ formula by hand. scipy exposes the same interface via the at= argument.

Matlab chebwin 直接吃 dB 衰減，不必手算 $\beta$。scipy 以 at= 對應相同介面。

% Matlab / Octave (Signal Processing Toolbox)
N     = 64;
A_dB  = 60;                     % desired side-lobe attenuation
w     = chebwin(N, A_dB);

% Spectrum check
W = abs(fftshift(fft(w, 1024)));
plot(20*log10(W / max(W)));     % should see flat ripple at -60 dB

# Python (scipy)
from scipy.signal.windows import chebwin
import numpy as np
N = 64
w = chebwin(N, at=60)

W = np.abs(np.fft.fftshift(np.fft.fft(w, 1024)))
# 20*log10(W / W.max()) → flat ripple at -60 dB

Typical $A$ values

常見 $A$ 值

$A = 40$ dB: very narrow main lobe, side-lobes at $-40$ dB — like a rectangular window, but controlled.
$A = 60$ dB: audio-grade baseline; comparable to Blackman but narrower main lobe.
$A = 80$ dB: precise spectrum analysis; comparable to Blackman-Harris.
$A = 100$-$120$ dB: near double-precision noise floor; useful for forensic analysis. $A = 40$ dB：主瓣很窄、旁瓣 $-40$ dB — 類似矩形視窗但受控。
$A = 60$ dB：音訊基準；類似 Blackman 但主瓣更窄。
$A = 80$ dB：精密頻譜分析；類似 Blackman-Harris。
$A = 100$-$120$ dB：逼近雙精度雜訊底；鑑識分析使用。

Matlab's chebwin actually computes the Dolph-Chebyshev formula for odd $N$ and interpolates for even $N$; very small even $N$ can show slight asymmetry. For audio $N \geq 32$ this is invisible.

Matlab chebwin 對奇數 $N$ 算真正的 Dolph-Chebyshev，偶數 $N$ 用內插；$N$ 很小的偶數會略不對稱。音訊 $N \geq 32$ 看不出來。

One function call, two parameters, exact minimax. Pick $N$ for main-lobe width, pick $A$ for side-lobe level — chebwin returns the provably optimal window.

一個函式、兩個參數、精確 minimax。$N$ 決定主瓣寬、$A$ 決定旁瓣高 — chebwin 回傳可證最優視窗。
