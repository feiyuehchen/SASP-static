Stage 4-9-4 · Kaiser & DPSS Windows Compared 關卡 4-9-4 · Kaiser 與 DPSS 視窗比較 SASP §4.9.4 ↗

Kaiser is a closed-form approximation to DPSS. In practice they agree to within a small fraction of a dB across the entire useful parameter range. Kaiser wins on implementation cost; DPSS wins only when you need the exact optimum or the full orthogonal taper family.

Kaiser 是 DPSS 的顯式近似。實務上兩者在整個可用範圍內差距不到 dB 的幾分之一。 Kaiser 實作便宜；只有在要精確最優或整組正交 taper 時才必須用 DPSS。

The mapping

對應關係

β ↔ NW
$$\beta\;\approx\;\pi\,NW\qquad\text{(both yield the same main-lobe / side-lobe trade)}$$

So DPSS with $NW=2$ ≈ Kaiser with $\beta\approx 2\pi\approx 6.28$. Both give about $-60$ dB side-lobes with a main-lobe a little over $2\pi NW/N$ wide.

所以 DPSS $NW=2$ ≈ Kaiser $\beta\approx 2\pi\approx 6.28$。兩者都給 $-60$ dB 左右旁瓣、主瓣寬略大於 $2\pi NW/N$。

When DPSS is worth the cost

何時值得用 DPSS

✓ Multi-taper spectrum estimation (Thomson's method) — needs the full orthogonal taper set.
✓ Provably optimal concentration — paper / thesis demands.
✓ Very small N (N < 16) — Kaiser's asymptotic formula starts to drift.
✗ One-off windowing in audio DSP — Kaiser is strictly easier and effectively identical.
✗ Real-time filter design — DPSS needs an eigendecomposition per design. ✓ 多 taper 頻譜估計（Thomson 法）— 需整組正交 taper。
✓ 可證最佳集中 — 論文 / 畢業口試要求。
✓ $N$ 非常小（$N < 16$）— Kaiser 漸近公式會漂移。
✗ 一次性音訊加窗 — Kaiser 絕對比較簡單、實際上相同。
✗ 即時濾波器設計 — DPSS 每組設計都要特徵分解。

Kaiser replaces DPSS for almost all audio work. Use $\beta=\pi NW$ to translate between them. Keep DPSS for multi-taper and theoretical proofs.

音訊工作幾乎都可用 Kaiser 取代 DPSS。用 $\beta=\pi NW$ 做換算。多 taper 與理論證明時才保留 DPSS。
