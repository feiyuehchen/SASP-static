% Matlab / Octave
n = (0:N-1)';
M = (N-1)/2;
alpha = 2;
hann_part    = 0.5 * (1 - cos(2*pi*n/(N-1)));
poisson_part = exp(-alpha * abs(n-M) / M);
w = hann_part .* poisson_part;

No built-in; build it by element-wise product. Sweep α to find the side-lobe level / main-lobe width combination that fits your problem.

沒有內建函式；逐元素相乘自己做。掃 α 找適合自己問題的旁瓣/主瓣組合。
