Stage 4-4-2 · Hamming-Windowed Oboe 關卡 4-4-2 · Hamming 加窗的雙簧管 SASP §4.4.2 ↗

The valley floor between harmonics drops to about $-42$ dB — we can now see a harmonic roughly $40$ dB quieter than the fundamental.

諧波之間的谷底下降到約 $-42$ dB — 比基頻弱 $40$ dB 左右的諧波現在看得到了。

What changed

發生了什麼

Same signal, same FFT size, only the window differs. Hamming's side-lobe floor at $-43$ dB is about $30$ dB lower than the rectangular's $-13$ dB. The first 8 harmonics at $200, 400, \ldots, 1600$ Hz with amplitudes $1, 0.6, \ldots, 0.01$ (span $40$ dB) now all emerge.

相同訊號、相同 FFT 長度，只換視窗。Hamming 的 $-43$ dB 旁瓣底比矩形的 $-13$ dB 低約 $30$ dB。在 $200, 400, \ldots, 1600$ Hz、振幅 $1, 0.6, \ldots, 0.01$（跨 $40$ dB）的前 8 諧波現在全部浮現。

Still close to the limit: the 8th harmonic at $-40$ dB barely clears the $-43$ dB floor. Any noise or a slightly stronger neighbour would bury it. For more headroom: Blackman (§4.4.3).

仍接近極限：第 8 諧波 $-40$ dB 勉強高於 $-43$ dB 底線。若有任何雜訊或鄰近強音稍變強就會淹沒它。要更多餘裕：用 Blackman（§4.4.3）。

Changing window shape = changing your measurement floor. Hamming converts the "can't see below $-13$ dB" problem into "can't see below $-43$ dB". Same signal, different visible reality.

改視窗形狀 = 改量測底線。Hamming 把「看不到 $-13$ dB 以下」變成「看不到 $-43$ dB 以下」。訊號相同、可見世界不同。
