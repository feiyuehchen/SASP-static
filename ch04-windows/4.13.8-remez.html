Grid LP discretises the stopband into thousands of points — accurate but heavy. Remez solves the same minimax problem using only $\lceil N/2\rceil+1$ extremal frequencies, updating them iteratively. Converges in 5-15 steps regardless of $N$.

格點 LP 把阻帶離散為數千點 — 精確但重。Remez 用 $\lceil N/2\rceil+1$ 個極值頻率就解同一個 minimax 問題，迭代更新。不論 $N$，5-15 步就收斂。

The alternation theorem

交替定理

Chebyshev's theorem: the optimal error function achieves its maximum magnitude at exactly $\lceil N/2\rceil+1$ points, with alternating sign. So finding the optimum = finding this alternating extremal set.

Chebyshev 定理：最優誤差函數在恰好 $\lceil N/2\rceil+1$ 個點達最大振幅、正負交替。所以求最優 = 找這組交替極值集合。

The loop

迴圈

1. Initialise extremal frequencies $\{\omega_1,\dots,\omega_M\}$ (uniform or Chebyshev nodes).
2. Solve small linear system: find $(w, \delta)$ such that $W(\omega_k)=\pm\delta$ with alternating signs.
3. Compute the resulting $W(\omega)$ on a dense grid.
4. Find all frequencies where $|W(\omega)|$ exceeds $|\delta|$ — these are new extrema.
5. Exchange: replace old extrema with new ones, preserving alternation.
6. Check convergence: $|\delta_{\text{new}} - \delta_{\text{old}}|$ small? Stop, else goto 2. 1. 初始化極值頻率 $\{\omega_1,\dots,\omega_M\}$（均勻或 Chebyshev 節點）。
2. 小線性系統求解：找 $(w, \delta)$ 使 $W(\omega_k)=\pm\delta$ 正負交替。
3. 密格點計算 $W(\omega)$。
4. 找所有 $|W(\omega)|$ 超過 $|\delta|$ 的頻率 — 為新極值。
5. 交換：新極值取代舊極值、保持交替。
6. 檢查收斂：$|\delta_{\text{new}} - \delta_{\text{old}}|$ 小？停止，否則回 2。

Comparison to grid LP

對照格點 LP

Grid LP: $N=1000$ with 4000 ω-points → $\sim 8000$ inequalities, seconds to solve.
Remez: same problem → $\sim 500$ extremal points per iteration, ~10 iterations, milliseconds total. 格點 LP：$N=1000$、4000 ω 點 → 約 8000 條不等式，數秒。
Remez：同問題 → 每次迭代約 500 極值，~10 次迭代，總共毫秒級。

Remez exchanges extrema, not variables. Leverages the alternation theorem to identify the handful of active constraints. The rest of the stopband is inactive by construction.

Remez 交換的是極值，非變數。利用交替定理辨識少數活躍約束。其餘阻帶依定義不活躍。
