Stage 4-13-4 · Positive-Sample Constraint 關卡 4-13-4 · 正樣本約束 SASP §4.13.4 ↗

Negative samples add oscillation to the taper shape (look at Dolph-Chebyshev at large $A$ — those end wiggles). Forcing $w[n]\ge 0$ gives a non-oscillating bell shape, which downstream processing often assumes (e.g. envelope analysis, STFT energy accumulation, visual pedagogy).

負樣本會讓形狀振盪（看大 $A$ 時的 Dolph-Chebyshev — 兩端的小翹曲）。強制 $w[n]\ge 0$ 得到不振盪的鐘形，下游處理常預設這個（如包絡分析、STFT 能量累計、視覺教學）。

THE CONSTRAINT
$$w[n]\;\ge\;0,\qquad n=0,\dots,N-1$$

These are $N$ simple lower bounds in the LP — each variable gets lb=0 in scipy, or a single linear inequality row $-w[n]\le 0$. Very cheap to add.

在 LP 中為 $N$ 個簡單下界 — scipy 中每個變數設 lb=0，或加一行 $-w[n]\le 0$。成本極低。

The cost

代價

Positivity restricts the feasible region, so the optimal $\delta$ grows. Typically:

正值約束限縮可行域，最優 $\delta$ 會變大。典型差異：

$A=60$ dB unconstrained → same main-lobe width ($\sim 5\pi/N$).
$A=60$ dB positive-only → main-lobe widens to $\sim 7\pi/N$ for the same $A$.
$A=100$ dB unconstrained → $\sim 7.9\pi/N$.
$A=100$ dB positive-only → won't achieve $A=100$ with compact support at small $N$. $A=60$ dB 無約束 → 主瓣寬 $\sim 5\pi/N$。
$A=60$ dB 僅正 → 同 $A$ 下主瓣擴至 $\sim 7\pi/N$。
$A=100$ dB 無約束 → $\sim 7.9\pi/N$。
$A=100$ dB 僅正 → 在小 $N$ 下無法達 $A=100$。

Hann, Hamming, Blackman, Kaiser, Gaussian, Poisson — all naturally positive. You only need this constraint if your formulation might produce negative tails (Dolph-Chebyshev, LP output).

Hann、Hamming、Blackman、Kaiser、Gaussian、Poisson — 天然皆正。只有當你的公式可能產生負尾（Dolph-Chebyshev、LP 輸出），才需要此約束。

Positivity is a design choice, not a property. Add it only when downstream processing needs it. Expect a few dB of attenuation or a modest main-lobe widening.

正值是設計選擇、非內稟性質。下游需要時才加。預期幾 dB 衰減損失或主瓣略寬。
