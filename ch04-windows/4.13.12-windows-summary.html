A complete mental catalogue of windows — from rectangular through Kaiser, Dolph-Chebyshev, Gaussian, DPSS, MLT sine, all the way to LP-designed custom shapes. And a feel for the four universal trade-offs: main-lobe width, first sidelobe level, asymptotic roll-off, reconstruction friendliness.

一份完整的視窗心智目錄 — 從矩形、Kaiser、Dolph-Chebyshev、Gaussian、DPSS、MLT 正弦、到 LP 設計的自訂形狀。以及對四個普世取捨的直覺：主瓣寬、第一旁瓣高、漸進滾降、重建友善度。

Five questions to pick a window

五個問題決定視窗

  1. Need reconstruction? COLA constraint → Hann, MLT sine, Vorbis, square-root-Hann.
  2. 需要重建？COLA 約束 → Hann、MLT 正弦、Vorbis、根 Hann。
  3. Where are interferers? Near → equiripple Chebyshev. Far → rolling-off Hann/Blackman/Kaiser. Targeted notch → LP with custom $g(\omega)$.
  4. 干擾在哪？近 → 等漣波 Chebyshev；遠 → 滾降 Hann/Blackman/Kaiser；目標陷波 → LP 加自訂 $g(\omega)$。
  5. Sidelobe target? Convert to dB. Kaiser β = 0.1102(A−8.7) for A>50; or chebwin(N, A) directly.
  6. 旁瓣目標？轉 dB。$A>50$ 時 Kaiser β = 0.1102(A−8.7)；或直接 chebwin(N, A)。
  7. Main-lobe target? Use the cheat card and pick the smallest $N$ that meets both axes.
  8. 主瓣目標？用對照表挑同時滿足兩軸的最小 $N$。
  9. Anything weird? (positivity, monotonicity, custom shape) → LP with the appropriate constraints.
  10. 有特殊需求？（正值、單調、自訂形狀）→ LP 配對應約束。

Window choice ≠ catalog lookup. Behind every classical window is a Pareto frontier — main-lobe width × sidelobe depth × roll-off rate. Closed-form windows are a coarse grid on that frontier; LP design fills in any specific point. The chapter taught you to read both.

選視窗 ≠ 查表。每個經典視窗背後都是一條柏拉圖前緣 — 主瓣寬 × 旁瓣深 × 滾降率。閉式視窗是前緣上的粗格；LP 設計可填入任一點。本章教會你讀懂兩者。

What's still possible

還能做什麼

  • Multi-criteria optimisation — minimise $\delta$ AND smoothness simultaneously (Pareto via weighted sum or ε-constraint).
  • 多目標最佳化 — 同時最小化 $\delta$ 與平滑度（加權和或 ε 約束的柏拉圖法）。
  • Asymmetric windows for sidelobe shaping in single-sided spectra (radar, sonar).
  • 不對稱視窗用於單邊頻譜旁瓣塑形（雷達、聲納）。
  • Time-varying windows that morph through a piece (Opus does this).
  • 時變視窗隨樂段變形（Opus 即如此）。
  • 2-D windows for image processing (separable products of any 1-D recipe in this chapter).
  • 2-D 視窗用於影像處理（本章任一 1-D 食譜的可分離乘積）。

World 5: FIR filter design. Same minimax + LP toolkit, different target — instead of a window with main-lobe and sidelobes, you'll design a passband / stopband filter with prescribed ripple. Parks-McClellan / Remez generalises to filter coefficients exactly the same way it generalises to window samples.

第 5 世界：FIR 濾波器設計。同樣的 minimax + LP 工具箱、不同目標 — 從主瓣＋旁瓣的視窗，變成有指定漣波的通帶／阻帶濾波器。Parks-McClellan/Remez 在濾波器係數上的廣義化方式與在視窗樣本上完全相同。
