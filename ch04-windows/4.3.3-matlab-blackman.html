Classic Blackman: $a_0=0.42,\; a_1=0.50,\; a_2=0.08$. Three numbers, every audio engineer knows them by heart.

經典 Blackman：$a_0=0.42,\; a_1=0.50,\; a_2=0.08$。三個數字，每個音訊工程師都背得出來。

% Matlab / Octave — built-in
w = blackman(N);

% Manual (exactly what blackman does)
n = (0:N-1)';
w = 0.42 - 0.5*cos(2*pi*n/(N-1)) + 0.08*cos(4*pi*n/(N-1));

The 0.08 is "exact enough"

0.08 已經「夠精確」

The truly optimal third coefficient for minimum first side-lobe is $0.07685$ (the exact Blackman). The traditional $0.08$ was chosen for ease of hand calculation. The penalty is only about $-58$ dB vs. $-68$ dB first side-lobe — a visible but not dramatic difference.

真正最佳化「最小第一旁瓣」的第三係數其實是 $0.07685$（精確 Blackman）。傳統選 $0.08$ 是為了手算方便。代價：第一旁瓣 $-58$ dB vs. $-68$ dB — 可見但不劇烈。

0.42 / 0.5 / 0.08 is a numerical mnemonic, not an optimum. Keep that in mind when the last few dB of side-lobe suppression matter.

0.42 / 0.5 / 0.08 是好記的整數組合，不是最佳解。需要最後幾 dB 旁瓣深度時別忘了這件事。
