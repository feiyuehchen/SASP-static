Matlab has hann(N) and hanning(N). They differ only at the endpoints — hann includes the zeros, hanning does not.

Matlab 提供 hann(N) 與 hanning(N)。差別只在端點：hann 含兩邊的零，hanning 不含。

% Matlab / Octave
w = 0.5 - 0.5 * cos(2*pi*(0:N-1)'/(N-1));

% Python / NumPy
w = 0.5 - 0.5 * np.cos(2*np.pi*np.arange(N) / (N-1))

% JavaScript (this site's DSP.js)
const w = DSP.WINDOWS.hann(N);

Any language. Any rate. Remember: the (N-1) in the denominator matters for symmetry. Using N instead will shift the window by half a sample and break COLA tricks.

任何語言、任何取樣率。注意：分母是 N-1，用 N 會讓視窗偏移半個樣本、破壞 COLA 的條件。
