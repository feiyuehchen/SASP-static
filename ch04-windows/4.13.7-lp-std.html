All the pieces from §4.13.1-6 combine into one LP tableau. Symmetry halves variables; DC anchors the solution; positivity is optional; $g(\omega)$ shapes the sidelobe ceiling.

§4.13.1-6 的所有片段合成一張 LP 總表。對稱減半變數；DC 錨定解；正值可選；$g(\omega)$ 塑造旁瓣天花板。

THE FULL LP
$$\begin{aligned} &\text{variables: } w[0],\ldots,w[\lceil N/2\rceil-1],\;\delta\\ &\min\;\delta\\ &\text{s.t.}\;\sum_n w[n]=1\quad\text{(DC)}\\ &\phantom{\text{s.t.}\;}\left|\sum_n w[n]\cos(\omega_i (n-\tfrac{N-1}{2}))\right|\le\delta\,g(\omega_i)\;\forall\omega_i\in\text{stopband}\\ &\phantom{\text{s.t.}\;}w[n]\ge 0\quad(\text{optional non-negativity}) \end{aligned}$$

The absolute value $|A w|\le b$ becomes two linear inequalities $Aw\le b$ and $-Aw\le b$. Feed to any solver with standard form $\min c^\top x$ s.t. $Ax\le b$, $A_{eq}x=b_{eq}$.

絕對值 $|A w|\le b$ 拆成兩條 $Aw\le b$ 與 $-Aw\le b$。餵給任何支援標準式 $\min c^\top x$ s.t. $Ax\le b$、$A_{eq}x=b_{eq}$ 的求解器。 

import numpy as np
from scipy.optimize import linprog

def design_window(N, omega_stop, g=None):
    """LP-design a window with custom sidelobe envelope.
    N           - length (odd for symmetric about centre)
    omega_stop  - grid of stopband frequencies
    g           - weight function values on that grid (None = flat)
    """
    M = (N+1)//2                           # half-window size (symmetric)
    g = np.ones_like(omega_stop) if g is None else np.asarray(g)
    n_sh = np.arange(M) - (N-1)/2          # shifted indices for zero-phase
    # Build cos basis for full-length symmetric spectrum
    # w[full] = mirror of w[half]; reconstruct cos(ω(n-mid)) for half + mirror
    C = np.cos(np.outer(omega_stop, n_sh))
    if N % 2 == 0:
        # ensure mirror pairs count correctly
        C[:,:-1] *= 2; # each pair contributes twice
    else:
        C[:,:-1] *= 2                       # middle sample singleton
    # inequalities C·w - δ·g ≤ 0 and -C·w - δ·g ≤ 0
    A_ub = np.vstack([
        np.hstack([C, -g[:,None]]),
        np.hstack([-C, -g[:,None]]),
    ])
    b_ub = np.zeros(2*len(omega_stop))
    # DC: Σ over full symmetric w = 1
    dc_row = np.zeros(M); dc_row[:-1] = 2; dc_row[-1] = 1
    A_eq = np.hstack([dc_row[None,:], [[0]]])
    b_eq = np.array([1.0])
    c = np.zeros(M+1); c[-1] = 1
    res = linprog(c, A_ub=A_ub, b_ub=b_ub, A_eq=A_eq, b_eq=b_eq,
                  bounds=[(None,None)]*M + [(0,None)], method='highs')
    half = res.x[:-1]
    w_full = np.concatenate([half, half[:-1][::-1]]) if N % 2 else \
             np.concatenate([half, half[::-1]])
    return w_full, res.x[-1]

One template, all named windows. Dolph-Chebyshev = flat $g$. Hann-like roll-off = $g \propto 1/\omega^2$. Custom = whatever $g$ your application needs. Change right-hand side, get new window.

一個模板、所有具名視窗。Dolph-Chebyshev = 平坦 $g$。類 Hann 滾降 = $g \propto 1/\omega^2$。客製 = 應用所需的 $g$。改右端 → 新視窗。
