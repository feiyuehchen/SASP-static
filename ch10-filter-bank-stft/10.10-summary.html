Stage 10-10 · World 10 Summary & Conclusions 關卡 10-10 · 第 10 世界總結與結論 SASP §10.10 ↗

The STFT matrix has two equivalent interpretations: rows = short-time spectra (OLA), columns = band-pass filter outputs (FBS). Perfect reconstruction in either interpretation is governed by one Poisson-summation-dual pair of conditions: COLA for OLA, Nyquist($N$) for FBS. Modifications inserted between analysis and synthesis must respect window bandwidth and frame-rate bounds or produce artefacts.

STFT 矩陣有兩種等價詮釋：列 = 短時頻譜（OLA）；行 = 帶通濾波器輸出（FBS）。任一詮釋下的完美重建皆受同一對 Poisson 求和對偶條件支配：OLA 要 COLA、FBS 要 Nyquist($N$)。在分析與合成間插入的修改必須尊重視窗頻寬與幀速率，否則產生失真。

Design decision tree

設計決策樹

Need time-varying effect? If no: use LTI FIR; skip STFT. If yes: continue.
Target side-lobe level? $-32$ dB → Hann, $-58$ dB → Blackman, $-74$ dB → Kaiser $\beta=8$ or Portnoff.
Latency budget? Small → minimise window length $M$; large → Portnoff long window + short FFT.
Hop $R$? Match to window's COLA family; typical $R=M/2$ (Hann) or $R=M/4$ (Kaiser).
WOLA or OLA? WOLA ($w_s=\sqrt{v}, w=\sqrt{v}$) for effect quality; OLA ($w_s=1$) for minimum CPU when modifications are gentle.
Mod rate? Keep $H_m[k]$ changes band-limited below half the frame rate $1/(2R)$. 需要時變效果嗎？否：用 LTI FIR 即可，略過 STFT。是：繼續。
目標旁波瓣位階？$-32$ dB → Hann、$-58$ dB → Blackman、$-74$ dB → Kaiser $\beta=8$ 或 Portnoff。
延遲預算？小 → 最小化視窗長度 $M$；大 → Portnoff 長視窗 + 短 FFT。
跳距 $R$？配合視窗的 COLA 家族；常見 $R=M/2$（Hann）或 $R=M/4$（Kaiser）。
WOLA 或 OLA？WOLA（$w_s=\sqrt{v}, w=\sqrt{v}$）給較好效果品質；OLA（$w_s=1$）在修改平滑時節省 CPU。
修改變化率？讓 $H_m[k]$ 變化頻寬低於幀速率一半 $1/(2R)$。

STFT FB = DFT + 視窗 + 跳距。第 10–12 世界其他內容都是這三個選擇的重參數化。把這三件事學好，其他自然落位。
