Stage 10-1-2 · Filter-Bank View of the STFT 關卡 10-1-2 · STFT 的濾波器組觀點 SASP §10.1.2 ↗

Show that column $k$ of the STFT matrix — the time history of bin $k$ — is exactly the output of a bandpass filter $h_k[n]$ applied to the input. We'll derive $h_k$ from the analysis formula, show $H_k(\omega)$ is a frequency-shifted copy of the window's spectrum, and see how the window's main-lobe width becomes the filter's bandwidth.

證明 STFT 矩陣第 $k$ 行 — 某個 bin 隨時間的變化 — 就是一個帶通濾波器 $h_k[n]$ 套在輸入上的輸出。接下來從分析公式導出 $h_k$、證明 $H_k(\omega)$ 是視窗頻譜的頻率位移版，並看視窗主波瓣寬度如何決定此濾波器的頻寬。

Starting point — stare at the analysis formula

起點 — 盯著分析公式

Keep $k$ fixed and let $m$ run. With the hop written as a continuous time index $n' = mR$:

固定 $k$，讓 $m$ 跑。把跳距寫成連續時間索引 $n' = mR$：

COLUMN k (BEFORE HOP-DECIMATION)
$$X_{n'}[k]\;=\;\sum_{n} x[n]\,w[n-n']\,e^{-j 2\pi k n / N}$$

Pull the exponential out of the sum using the shift $n = (n-n') + n'$:

用替換 $n = (n-n') + n'$ 把指數拆開：

$$X_{n'}[k]\;=\;e^{-j 2\pi k n' / N}\sum_{n} x[n]\,w[n-n']\,e^{-j 2\pi k (n-n') / N}$$

The sum has the shape of a convolution. Define the filter:

求和寫成卷積的樣子。定義濾波器：

CHANNEL k IMPULSE RESPONSE
$$h_k[n]\;\triangleq\;w[-n]\,e^{j 2\pi k n / N}$$

Then column $k$ is literally convolution times a phase factor:

於是第 $k$ 行就是「卷積 × 相位因子」：

$$X_{n'}[k]\;=\;e^{-j 2\pi k n' / N}\,(x * h_k)[n']$$

That leading exponential is the only difference between the STFT bin and a straight bandpass output. It's often absorbed into the synthesis (gives zero-phase at $n=0$), or dropped when we only care about $|X_{n'}[k]|^2$ (spectrograms).

前面那個複指數是 STFT bin 與純帶通輸出之間唯一的差別。通常會被合成端吸收（使 $n=0$ 時相位為零），或在只看 $|X_{n'}[k]|^2$（聲譜圖）時直接忽略。

What does $H_k(\omega)$ look like?

$H_k(\omega)$ 長什麼樣？

Take the DTFT of $h_k[n] = w[-n]\,e^{j 2\pi k n / N}$. Time-reversal conjugates the spectrum; modulation shifts it:

對 $h_k[n] = w[-n]\,e^{j 2\pi k n / N}$ 取 DTFT。時間反轉共軛頻譜、乘複指數平移頻譜：

CHANNEL k FREQUENCY RESPONSE
$$H_k(\omega)\;=\;W^*\!\left(\omega_k - \omega\right)\;=\;W(\omega - \omega_k)\Big|_{\text{for real }w},\quad \omega_k\triangleq\tfrac{2\pi k}{N}$$

So every channel is a shifted copy of the same prototype lowpass $W(\omega)$, just re-centred to $\omega_k$. The STFT is a uniformly modulated filter bank.

所以每個通道都是同一個低通原型 $W(\omega)$ 的頻率位移版，只差把中心平移到 $\omega_k$。STFT 就是一個「均勻調變濾波器組」。

Window shape → filter quality. Main-lobe width of $W$ ⇒ channel bandwidth.
Side-lobe level of $W$ ⇒ out-of-band rejection between channels.
Window length $M$ ⇒ filter length (so longer windows = narrower but slower filters).

視窗形狀 → 濾波器品質。 $W$ 的主波瓣寬 ⇒ 通道頻寬。
$W$ 的旁波瓣位階 ⇒ 通道間的外頻率抑制。
視窗長度 $M$ ⇒ 濾波器長度（視窗越長 → 越窄但越慢）。

FB view is frequency-first. Each bin is a bandpass-filtered version of the input. This is the right lens for equalisers, modal analysis, source separation, and anything where you want to reason "what happens to this frequency?" locally, without thinking about frames.

FB 觀點「頻率優先」。每個 bin 就是輸入被帶通濾波後的結果。這是等化器、模態分析、聲源分離，以及任何「就這個頻率想它怎麼變」的情境最自然的觀點 — 完全不必想 frame。
