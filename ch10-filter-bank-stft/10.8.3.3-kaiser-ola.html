Stage 10-8-3-3 · Kaiser OLA Example 關卡 10-8-3-3 · Kaiser OLA 範例 SASP §10.8.3.3 ↗

Kaiser is an infinite-support window approximated by a truncated Bessel — PSF can't hit exactly zero like it did for Hamming. Quantify the approximate-COLA ripple as a function of $\beta$ and $R/M$, and show which $(\beta, R)$ pairs beat $-80$ dB in practice.

Kaiser 是無窮支撐視窗的截斷 Bessel 近似 — PSF 無法像 Hamming 那樣恰為零。本節以 $\beta$ 與 $R/M$ 為變數量化近似 COLA 的漣波，展示哪些 $(\beta, R)$ 對能在實務上達 $-80$ dB。

The Kaiser window

Kaiser 視窗

$$w_{\text{Kaiser}}[n]\;=\;\frac{I_0\!\left(\beta\sqrt{1 - (2n/M - 1)^2}\right)}{I_0(\beta)},\quad 0\le n < M$$

$I_0$ is the modified Bessel of the first kind. $\beta$ controls main-lobe/side-lobe trade-off; common values: $\beta=6$ ($-60$ dB side-lobes), $\beta=8$ ($-74$ dB), $\beta=10$ ($-90$ dB).

$I_0$ 為第一類修正貝索函數。$\beta$ 控制主/旁波瓣取捨；常見值：$\beta=6$（$-60$ dB）、$\beta=8$（$-74$ dB）、$\beta=10$（$-90$ dB）。

COLA ripple — numerical approach

COLA 漣波 — 數值路徑

Unlike Hamming, Kaiser's DTFT has a continuous main lobe with rapid side-lobe decay. For hop $R$, PSF sampling at $\omega = 2\pi k/R$ picks up small-but-non-zero values for $k\ne 0$:

與 Hamming 不同，Kaiser 的 DTFT 是連續主波瓣 + 快速衰減旁波瓣。跳距 $R$ 下，在 $\omega = 2\pi k/R$ 取樣時 $k\ne 0$ 處有小但非零的值：

COLA RIPPLE
$$\text{ripple}\;\approx\;\frac{2}{W(0)}\sum_{k=1}^{\infty}\left|W\!\left(\tfrac{2\pi k}{R}\right)\right|\cos\!\left(\tfrac{2\pi k n}{R}\right)$$

In practice: for audio work, $\beta=8$ at $R=M/4$ is a sweet spot ($-72$ dB ripple is imperceptible). If you need machine-precision PR for testing, push to $\beta=10$ at $R=M/8$.

實務：音訊工作常用 $\beta=8$、$R=M/4$（$-72$ dB 漣波無法聽出）。若測試需要機器精度 PR，拉到 $\beta=10$、$R=M/8$。

Engineering COLA = pick an $\epsilon$ and chase it. Kaiser's tunable $\beta$ lets you trade the main-lobe width for the COLA ripple, giving a continuous design knob that Hamming/Hann don't have.

工程版 COLA = 選一個 $\epsilon$ 然後追它。Kaiser 的可調 $\beta$ 讓你在主波瓣寬與 COLA 漣波間連續取捨，這是 Hamming/Hann 沒有的設計旋鈕。
